# Contributing

This repository contains the public protocol specification for REDAX Hub. Implementation code is not included here.

## What we welcome

- Issues reporting ambiguity, contradiction, or missing detail in the specification
- Suggestions for clarification, especially around invariants and threat vectors
- Documentation improvements (typos, formatting, cross-references)

## What is out of scope

- Implementation patches (code lives in a private repository until audit completion)
- Phase 1 feature requests (specification is in freeze)
- Phase 2 designs without prior discussion

## How to contribute

1. Open an issue describing the problem or suggestion
2. Wait for triage by the REDAX team
3. If approved, submit a pull request referencing the issue

## Code of conduct

Be technical. Be precise. Disagree on substance, not style.

## Contact

- Specification questions: open an issue
- Security concerns: see SECURITY.md
- General contact: hello@redaxhub.com
