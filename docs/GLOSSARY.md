# Glossary

- **Campaign PDA**: Program-derived account that holds campaign state and acts as bounded mint authority.
- **campaign_admin**: Operational authority for pause, finalize, and operational updates.
- **output_governance**: Economic owner of the output token and metadata update authority.
- **protocol_fee_vault**: PDA that holds the protocol fee (1% output tokens).
- **project_treasury**: Off-protocol treasury controlled by `output_governance`.
- **treasury_policy**: Locked execution policy for `protocol_fee_vault`.
- **merger_type**: Campaign type enum on Campaign PDA.
- **output_mint_mode**: Output mint mode enum on Campaign PDA.
- **metadata_strategy**: Output metadata strategy enum on Campaign PDA.
- **LegacyProjectAttestation**: On-chain proof that a legacy project endorsed a campaign.
- **authority_policy**: Post-finalization mint authority policy.
- **Discovery Policy**: Phase 1 UI-layer rule for curated discovery and warnings.
- **Verified Tier**: Phase 2 governance layer for campaign review.
